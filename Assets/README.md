# Usage

## Java Game Server
1. (optional) To rebuild Java project:
    #### `cd game-server`

    Build using gradle:
    #### `gradle build`

2. Running Java server
   #### `cd game-server/build/libs`

   Execute `.jar` file (Use appropriate file name)
   #### `java -jar game-server-*.*.*-SNAPSHOT.jar`

## Unity Game
1. Build and/or run Unity project as usual
